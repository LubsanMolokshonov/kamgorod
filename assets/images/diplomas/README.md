# Шаблоны дипломов

## Инструкция

Поместите изображения шаблонов дипломов в соответствующие папки:

### templates/
Полноразмерные изображения шаблонов (рекомендуемый размер: A4 landscape - 297x210mm или 3508x2480px в 300 DPI)

Пример файлов:
- `template1.jpg` - Классический синий
- `template2.jpg` - Элегантный фиолетовый
- `template3.jpg` - Современный градиент

### thumbnails/
Миниатюры для предпросмотра (рекомендуемый размер: 400x283px)

Пример файлов:
- `template1_thumb.jpg`
- `template2_thumb.jpg`
- `template3_thumb.jpg`

## Временное решение

Пока вы не добавили собственные шаблоны, система использует placeholder изображения.
Вы можете найти бесплатные шаблоны дипломов на:
- Canva.com
- Freepik.com
- Pinterest.com (поиск "certificate template")

## Формат изображений

- Формат: JPG или PNG
- Ориентация: Альбомная (landscape)
- Качество: Высокое (для печати - 300 DPI)
