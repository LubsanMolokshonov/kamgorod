# Шаблоны дипломов

Коллекция из 6 профессиональных шаблонов дипломов для конкурсов.

## Описание шаблонов

### Диплом 1 - Классический голубой
- **Цветовая схема**: Синий/Голубой (#3b82f6)
- **Стиль**: Современный, классический
- **Особенности**: Градиентный фон, золотая звезда, четкие границы
- **Подходит для**: Всероссийские конкурсы, академические достижения

### Диплом 2 - Золотой победитель
- **Цветовая схема**: Золотой/Желтый (#f59e0b)
- **Стиль**: Торжественный, праздничный
- **Особенности**: Звезда вверху, большая золотая медаль, бейдж "I МЕСТО"
- **Подходит для**: Победители, 1 место, международные конкурсы

### Диплом 3 - Голубая медаль
- **Цветовая схема**: Небесно-голубой (#0ea5e9)
- **Стиль**: Элегантный, спортивный
- **Особенности**: Медаль с лентами, округлые формы
- **Подходит для**: 2 место, спортивные достижения

### Диплом 4 - Фиолетовый щит
- **Цветовая схема**: Фиолетовый/Пурпурный (#9333ea)
- **Стиль**: Благородный, академический
- **Особенности**: Угловые украшения, щит с номером места
- **Подходит для**: 3 место, творческие конкурсы

### Диплом 5 - Красный престиж
- **Цветовая схема**: Красный/Алый (#dc2626)
- **Стиль**: Престижный, официальный
- **Особенности**: Розетка вверху с лентами, большой круг с номером места
- **Подходит для**: Международные конкурсы, особые достижения

### Диплом 6 - Зеленый эко
- **Цветовая схема**: Зеленый/Изумрудный (#10b981)
- **Стиль**: Свежий, экологический
- **Особенности**: Лавровый венок, кубок, рамка "УЧАСТНИК"
- **Подходит для**: Экологические проекты, участие в конкурсах

## Технические характеристики

- **Формат**: SVG (масштабируемая векторная графика)
- **Размер**: 600x848 пикселей (формат A4 в портретной ориентации)
- **Расположение**:
  - `/assets/images/diplomas/templates/` - полноразмерные шаблоны
  - `/assets/images/diplomas/thumbnails/` - миниатюры для галереи

## Использование

Шаблоны автоматически отображаются в форме регистрации на конкурс. Пользователь может:

1. Просматривать миниатюры всех 6 шаблонов в галерее
2. Кликать на миниатюру для предпросмотра полноразмерного диплома
3. Выбранный шаблон будет использован при генерации финального диплома

## Персонализация

Каждый шаблон содержит текстовые поля-примеры, которые будут заменены на реальные данные:
- ФИО участника
- Название конкурса (Всероссийский/Международный/Межрегиональный)
- Номинация
- Название работы
- Место (1, 2, 3 или Участник)
- Учебное учреждение
- Руководитель (если есть)
- Дата участия

## Обновление шаблонов

Для добавления нового шаблона:

1. Создайте SVG файл в формате 600x848px
2. Сохраните как `diploma-template-N.svg` в обе директории:
   - `/assets/images/diplomas/templates/`
   - `/assets/images/diplomas/thumbnails/`
3. Обновите `registration.php` для отображения нового шаблона
