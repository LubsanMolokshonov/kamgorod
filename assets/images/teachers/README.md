# Изображения педагогов для главной страницы

Эта папка содержит изображения педагогов, которые отображаются на первом экране лендинга.

## Текущие файлы

В настоящее время используются SVG-заглушки (placeholder):
- `teacher-1.svg` - розовый фон
- `teacher-2.svg` - фиолетовый фон
- `teacher-3.svg` - оранжевый фон
- `teacher-4.svg` - голубой фон
- `teacher-5.svg` - сиреневый фон

## Как заменить на реальные фотографии

1. Подготовьте 5 фотографий педагогов в формате JPG или PNG
2. Рекомендуемые параметры:
   - Размер: минимум 500x500 пикселей (квадратные)
   - Формат: JPG (оптимальный баланс качества и размера)
   - Качество: 85-90%
   - Важно: фотографии должны быть портретными, с хорошо видимым лицом

3. Переименуйте файлы:
   - `teacher-1.jpg`
   - `teacher-2.jpg`
   - `teacher-3.jpg`
   - `teacher-4.jpg`
   - `teacher-5.jpg`

4. Загрузите файлы в эту папку (заменив SVG файлы)

5. Обновите пути в файле `/index.php`:
   ```php
   Замените .svg на .jpg в строках:
   - /assets/images/teachers/teacher-1.svg → /assets/images/teachers/teacher-1.jpg
   - /assets/images/teachers/teacher-2.svg → /assets/images/teachers/teacher-2.jpg
   - и т.д.
   ```

## Рекомендации по подбору фотографий

- Используйте профессиональные фотографии педагогов
- Убедитесь, что у вас есть разрешение на использование этих фотографий
- Фотографии должны быть разнообразными (разные возраста, пол)
- Желательно, чтобы педагоги на фото выглядели дружелюбно и профессионально
- Фон на фотографиях должен быть нейтральным или размытым

## Альтернативные источники изображений

Если у вас нет собственных фотографий, можно использовать:
- Стоковые фотографии с сайтов: Unsplash, Pexels (бесплатные)
- Сгенерированные AI изображения (убедитесь в соблюдении авторских прав)
- Фотографии коллег с их письменного согласия
